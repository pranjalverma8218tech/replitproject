Requirements:

1. Language Switcher Visibility
- The language switcher must be globally visible on every page.
- Place it in the Header/Navbar.
- Make it clearly visible from a distance.
- Use a prominent button or dropdown with:
  🇮🇳 हिन्दी | 🇬🇧 English
- Keep it sticky so it remains visible while scrolling.
- It should appear on:
  - Home Page
  - Category Pages
  - Product Pages
  - Cart Page
  - Checkout / Buy Now
