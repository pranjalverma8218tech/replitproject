ADMIN PANEL → HOMEPAGE MANAGER
==============================

Create a dedicated Homepage Manager.

Each section must have its own edit panel.

* Hero Image

* Hero Background Image
* Title
* Subtitle

* Add Category

* Upload Category Image
* Change Category Name
* Reorder Categories

IMAGE REQUIREMENTS
==================

For every image:

* Upload
* Replace
* Delete
* Preview
* Drag & Drop Reorder

=================================
DATABASE
========

Store all homepage content in database.

Do not use hardcoded images.

All homepage content must be loaded dynamically from admin-managed data.

=================================
RESULT
======

Admin should be able to completely redesign the homepage without touching code.

Every image, icon, banner, review photo, category image and CTA graphic must be manageable from Admin Panel.
